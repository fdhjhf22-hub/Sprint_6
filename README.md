# Sprint 6 – Финальный проект по автотестированию сервиса Яндекс.Самокат

## Описание
Проект содержит UI-автотесты для учебного сервиса [Яндекс.Самокат](https://qa-scooter.praktikum-services.ru/). Реализована модель Page Object, параметризация тестов и генерация Allure-отчётов.

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone git@github.com:fdhjhf22-hub/Sprint_6.git
   cd Sprint_6

2. Создайте и активируйте виртуальное окружение:

python -m venv venv
source venv/Scripts/activate      # для Git Bash / Linux / Mac
# или
venv\Scripts\activate             # для CMD / PowerShell

3. Установите зависимости:

pip install -r requirements.txt

4. Запустите тесты с генерацией Allure-отчёта:

pytest --alluredir=allure-results

5. Просмотрите отчёт:

allure serve allure-results

Структура проекта

    locators/ – локаторы элементов страниц

    pages/ – классы Page Object

    tests/ – тесты, разделённые по функциональности

    data.py – тестовые данные

    urls.py – константы URL

    conftest.py – фикстура драйвера Firefox

Реализованные тесты

    Проверка выпадающих ответов в разделе «Вопросы о важном» (8 вопросов, параметризация)

    Позитивный сценарий заказа самоката (2 набора данных, параметризация)

    Проверка перехода по логотипу Самоката на главную страницу

    Проверка открытия Дзена в новой вкладке по клику на логотип Яндекса
